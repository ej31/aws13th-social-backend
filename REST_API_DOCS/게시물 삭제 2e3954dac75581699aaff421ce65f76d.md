# 게시물 삭제

리소스: Posts
method: DELETE
API path: posts, {post_id}
Error: 401, 422
비고: 본인이 작성한 게시물을 삭제한다. 본인 작성 글만 삭제 가능하다.

### 게시물 삭제

**DELETE** `/posts/{post_id}`

본인이 작성한 게시물을 삭제한다. 본인 작성 글만 삭제 가능하다.

**Request Headers**

| 헤더 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| Authorization | string | ✅ | 게시물 삭제를 하는 사람이 본인임을 확인하기 위한 토큰 (`Bearer <access_token>` 형식의 인증 토큰)(예: `Authorization: Bearer eyJ...`) |

**Path Parameters**

| 파라미터 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| `post_id` | string | ✅ | 삭제할 게시물의 고유 ID |

**Request Example** 

```
DELETE /posts/post_02
```

**Response (204 No Content)**

```
(no response body)
```

**Error**

| HTTP | code | message | 발생 조건 |
| --- | --- | --- | --- |
| 401 | AUTH_401_01 | Authorization required. | Authorization 헤더 없음 |
| 401 | AUTH_401_02 | Invalid token. | 토큰 위조/형식 오류 |
| 401 | AUTH_401_03 | Token expired. | 토큰 만료 |
| 422 | POSTS_422_07 | Invalid post_id. | post_id 형식 오류 |
| 404 | POSTS_404_01 | Post not found. | 해당 post_id 게시글 없음 |
| 403 | POSTS_403_01 | Forbidden. | 작성자 본인이 아님 |
| 500 | COMMON_500_01 | Internal server error. | 서버 내부 오류 |

### 내가 쓴 게시물 목록

**GET** `/users/me/posts` 

내가 쓴 게시글 목록을 살펴본다.

**Request Headers**

| 헤더 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| Authorization | string | ✅ | 게시물 조회를 하는 사람이 본인임을 확인하기 위한 토큰 (`Bearer <access_token>` 형식의 인증 토큰)(예: `Authorization: Bearer eyJ...`) |

**Query Parameters**

| 파라미터 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| page | integer | ❎ | 페이지 번호 (기본값 : 1) |
| limit | integer | ❎ | 페이지 당 개수 (기본값 : 20, 최대 100) |
| q | string | ❎ | 검색어 (제목, 내용) |
| search_by | string | ❎ | 검색 범위( title | content |  title,content)
q가 있을때 적용
기본값: title,content |
| sort | string | ❎ | 정렬 기준(created_at | view_count | like_count) |
| order | string | ❎ | 정렬 방향 (asc, desc(기본값))
sort와 함께 사용한다 |

**Error**

| HTTP | code | message | 발생 조건 |
| --- | --- | --- | --- |
| 401 | AUTH_401_01 | Authorization required. | Authorization 헤더 없음 |
| 401 | AUTH_401_02 | Invalid token. | 토큰 위조/형식 오류 |
| 401 | AUTH_401_03 | Token expired. | 토큰 만료 |
| 422 | POSTS_422_01 | Invalid page. | page < 1 또는 숫자 아님 |
| 422 | POSTS_422_02 | Invalid limit. | limit < 1 또는 limit > 100 또는 숫자 아님 |
| 422 | POSTS_422_03 | Invalid sort. | sort ∉ {created_at, view_count, like_count} |
| 422 | POSTS_422_04 | Invalid order. | order ∉ {asc, desc} |
| 422 | POSTS_422_05 | Invalid search_by. | search_by ∉ {title, content, title,content} |
| 422 | POSTS_422_06 | q required for search_by. | search_by를 줬는데 q가 없음 |
