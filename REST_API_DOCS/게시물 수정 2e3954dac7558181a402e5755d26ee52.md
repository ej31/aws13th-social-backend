# 게시물 수정

리소스: Posts
method: PATCH
API path: posts, {post_id}
Error: 400, 401, 403, 404, 413, 415, 422
비고: 본인이 작성한 게시물을 수정합니다.

### 게시물 수정

**PATCH** `/posts/{post_id}`

본인이 작성한 게시물을 수정합니다.

**Request Headers**

| 헤더 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| Content-Type | multipart/form-data | ✅ | 텍스트 필드와 파일(**첨부파일**)을 함께 전송하기 위한 포맷 |
| Authorization | string | ✅ | 게시물 수정을 하는 사람이 본인임을 확인하기 위한 토큰 (`Bearer <access_token>` 형식의 인증 토큰)(예: `Authorization: Bearer eyJ...`) |

**Path Parameters**

| 파라미터 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| `post_id` | string | ✅ | 수정할 게시물의 고유 ID |

**Request Body**

| 필드 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| title | string | ❎ | 수정할 게시물의 제목 |
| content | string | ❎ | 수정할 게시물의 본문 |
| post_file | file | ❎ | 첨부파일(선택). 허용 형식: jpeg/png/mp3/mp4. 미전송 시 기존 첨부파일은 유지된다. |

**Request Example  (multipart/form-data)**

| title | 수정할 게시물 제목 |
| --- | --- |
| content | 수정할 게시물 내용 |
| post_file | `post.png` ,`post.jpeg` , `post.mp3` ,`post.mp4`      |

**Response (200 OK)**

```json
{
  "status": "success",
  "data": {
    "id": "post_02",
    "title": "수정한 게시물 제목 ",
    "content": "수정한 게시물 내용",
    "view_count": 100,
    "like_count": 10,
    "attachment_url": "https://.../post_02/file.mp4",
    "created_at": "2026-01-04T12:00:00Z",
    "updated_at" : "2026-01-05T12:00:00Z",
    "author": {
      "id": "user_1234567abc",
      "nickname": "sihwa",
      "profile_image_url": "https://api.example.com/uploads/profiles/default.png"
    }
  }
}
```

**Error**

| HTTP | code | message | 발생 조건 |
| --- | --- | --- | --- |
| 400 | COMMON_400_02 | Multipart parse failed. | multipart 파싱 실패 |
| 401 | AUTH_401_01 | Authorization required. | Authorization 헤더 없음 |
| 401 | AUTH_401_02 | Invalid token. | 토큰 위조/형식 오류 |
| 401 | AUTH_401_03 | Token expired. | 토큰 만료 |
| 403 | POSTS_403_01 | Forbidden. | 작성자 본인이 아님 |
| 404 | POSTS_404_01 | Post not found. | 해당 post_id 게시글 없음 |
| 413 | UPLOAD_413_02 | File too large. | post_file 용량 제한 초과 |
| 415 | UPLOAD_415_02 | Unsupported file type. | post_file 타입 제한 위반 |
| 415 | COMMON_415_01 | Unsupported media type. | Content-Type이 multipart/form-data 아님 |
| 422 | POSTS_422_08 | Invalid title. | title 정책 위반 |
| 422 | POSTS_422_09 | Invalid content. | content 정책 위반(빈값 등) |
| 422 | POSTS_422_07 | Invalid post_id. | post_id 형식 오류 |
