# 내 프로필 조회

리소스: Users
method: GET
API path: me, users
Error: 401, 404
비고: 로그인 한 이후 사용자 본인의 정보를 조회한다.

### 내 프로필 조회

**GET** `/users/me` 

로그인 한 이후 사용자 본인의 정보를 조회한다. 

**Request Headers**

| 헤더 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| Authorization | string | ✅ | 로그인한 본인을 확인하기 위한 토큰 (Bearer 스킴) |

**Response (200 OK)**

```json
{
  "status": "success",
  "data": 
  {
    "id": "user_1234567abc",
    "email": "sigwan116@gmail.com",
    "nickname": "sihwa",
    "profile_image_url": "https://api.example.com/uploads/profiles/default.png",
    "created_at": "2026-01-04T12:00:00Z"
  }
}

```

**Error**

| HTTP | code | 언제 나옴 |
| --- | --- | --- |
| 401 | AUTH_401_01 | 토큰 없음 |
| 401 | AUTH_401_02 | 토큰 유효하지 않음 |
| 401 | AUTH_401_03 | 토큰 만료 |
| 404 | USERS_404_02 | 토큰은 유효한데 해당 유저가 삭제됨/없음(선택) |
