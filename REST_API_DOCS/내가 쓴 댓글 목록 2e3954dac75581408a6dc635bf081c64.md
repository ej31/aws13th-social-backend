# 내가 쓴 댓글 목록

리소스: Comments
method: GET
API path: comments, me, users
Error: 401, 403, 404, 422
비고: 내가 쓴 댓글 목록을 살펴본다.

### 내가 쓴 댓글 목록

**GET** `/users/me/comments` 

내가 쓴 댓글 목록을 살펴본다.

**Request Headers**

| 헤더 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| Authorization | string | ✅ | 댓글 조회를 하는 사람이 본인임을 확인하기 위한 토큰 (`Bearer <access_token>` 형식의 인증 토큰)(예: `Authorization: Bearer eyJ...`) |

**Query Parameters**

| 파라미터 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| page | integer | ❎ | 페이지 번호 (기본값 : 1) |
| limit | integer | ❎ | 페이지 당 개수 (기본값 : 20, 최대 100) |
| q | string | ❎ | 검색어 (댓글 내용에서 검색) |
| search_by | string | ❎ | 검색 범위 :  content(고정값) |
| sort | string | ❎ | 정렬 기준 (created_at) (기본값: created_at) |
| order | string | ❎ | 정렬 방향 (asc, desc(기본값))
sort와 함께 사용한다 |

**Response (200 OK)**

```json
{
  "status": "success",
  "data": [
    {
      "id": "comment_01",
      "post": {
        "id": "post_01",
        "title": "게시물 제목 1"
      },
      "content": "댓글 내용 1",
      "created_at": "2026-01-04T12:00:00Z"
    },
    {
      "id": "comment_02",
      "post": {
        "id": "post_09",
        "title": "게시물 제목 9"
      },
      "content": "댓글 내용 2",
      "created_at": "2026-01-05T12:00:00Z"
    }
  ],
  "pagination": {
    "page": 1,
    "limit": 20,
    "total": 100
  }
}
```

**Error**

| HTTP | code | message | 발생 조건 |
| --- | --- | --- | --- |
| 401 | AUTH_401_01 | Authorization required. | `Authorization` 헤더 없음 |
| 401 | AUTH_401_02 | Invalid token. | 토큰 위조/형식 오류 |
| 401 | AUTH_401_03 | Token expired. | 토큰 만료 |
| 403 | COMMENTS_403_01 | Forbidden. | 작성자 본인이 아님 |
| 404 | POSTS_404_01 | Post not found. | 해당 `post_id` 게시글 없음 |
| 404 | COMMENTS_404_01 | Comment not found. | 해당 `comment_id` 댓글 없음 |
| 422 | COMMENTS_422_08 | Invalid comment_id. | `comment_id` 형식 오류 |
| 422 | POSTS_422_07 | Invalid post_id. | `post_id` 형식 오류 |
