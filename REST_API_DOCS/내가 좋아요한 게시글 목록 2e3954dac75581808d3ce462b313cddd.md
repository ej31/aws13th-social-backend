# 내가 좋아요한 게시글 목록

리소스: Likes
method: GET
API path: likes, me, users
Error: 401, 422
비고: 내가 좋아요한 게시글 목록을 살펴본다.

### 내가 좋아요한 게시글 목록

**GET** `/users/me/likes` 

내가 좋아요한 게시글 목록을 살펴본다.

**Request Headers**

| 헤더 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| Authorization | string | ✅ | 좋아요한 게시글 목록 조회 요청자(본인)  확인하기 위한 토큰 (`Bearer <access_token>` 형식의 인증 토큰)(예: `Authorization: Bearer eyJ...`) |

**Query Parameters**

| 파라미터 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| page | integer | ❎ | 페이지 번호 (기본값 : 1) |
| limit | integer | ❎ | 페이지 당 개수 (기본값 : 20, 최대 100) |
| q | string | ❎ | 검색어 (제목, 내용) |
| search_by | string | ❎ | 검색 범위( title | content |  title,content)
q가 있을때 적용
기본값: title,content |
| sort | string | ❎ | 정렬 기준(created_at | view_count | like_count | liked_at) |
| order | string | ❎ | 정렬 방향 (asc, desc(기본값))
sort와 함께 사용한다 |

  "data": [

        "author": {

**Error**

| HTTP | code | message | 발생 조건 |
| --- | --- | --- | --- |
| 401 | AUTH_401_01 | Authorization required. | `Authorization` 헤더 없음 |
| 401 | AUTH_401_02 | Invalid token. | 토큰 위조/형식 오류 |
| 401 | AUTH_401_03 | Token expired. | 토큰 만료 |
| 422 | LIKES_422_01 | Invalid page. | `page < 1` 또는 숫자 아님 |
| 422 | LIKES_422_02 | Invalid limit. | `limit < 1` 또는 `limit > 100` 또는 숫자 아님 |
| 422 | LIKES_422_03 | Invalid sort. | `sort ∉ {created_at, view_count, like_count, liked_at}` |
| 422 | LIKES_422_04 | Invalid order. | `order ∉ {asc, desc}` |
| 422 | LIKES_422_05 | Invalid search_by. | `search_by ∉ {title, content, title,content}` |
| 422 | LIKES_422_06 | q required for search_by. | `search_by`를 줬는데 `q`가 없음 |
