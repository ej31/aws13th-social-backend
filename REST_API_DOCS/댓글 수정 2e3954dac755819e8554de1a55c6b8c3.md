# 댓글 수정

리소스: Comments
method: PATCH
API path: comments, posts, {comment_id}, {post_id}
Error: 400, 401, 403, 404, 405, 422
비고: 본인이 작성한 댓글을 수정합니다.

### 댓글 수정

**PATCH** `/posts/{post_id}/comments/{comment_id}`

본인이 작성한 댓글을 수정합니다.

**Request Headers**

| 헤더 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| Content-Type | application/json | ✅ | 텍스트를 전송하기 위한 포맷 |
| Authorization | string | ✅ | 댓글 수정을 하는 사람이 본인임을 확인하기 위한 토큰 (`Bearer <access_token>` 형식의 인증 토큰)(예: `Authorization: Bearer eyJ...`) |

**Path Parameters**

| 파라미터 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| `post_id` | string | ✅ | 댓글이 속한 게시글의 고유 ID |
| `comment_id` | string | ✅ | 수정할 댓글의 고유 ID |

**Request Body**

| 필드 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| content | string | ✅ | 수정할 댓글의 내용 |

**Request Example** 

```json
{
  "content": "댓글 수정 내용 1"
}
```

**Response (200 OK)**

```json
{
  "status": "success",
  "data": {
    "id": "comment_01",
    "post_id": "post_01",
    "content": "수정한 댓글 내용 1",
    "created_at": "2026-01-04T12:00:00Z",
    "updated_at": "2026-01-05T12:00:00Z",
    "author": {
      "id": "user_1234567abc",
      "nickname": "sihwa",
      "profile_image_url": "https://api.example.com/uploads/profiles/default.png"
    }
  }
}
```

**Error**

| HTTP | code | message | 발생 조건 |
| --- | --- | --- | --- |
| 400 | COMMON_400_01 | JSON parse failed. | JSON 파싱 실패 |
| 401 | AUTH_401_01 | Authorization required. | `Authorization` 헤더 없음 |
| 401 | AUTH_401_02 | Invalid token. | 토큰 위조/형식 오류 |
| 401 | AUTH_401_03 | Token expired. | 토큰 만료 |
| 403 | COMMENTS_403_01 | Forbidden. | 작성자 본인이 아님 |
| 404 | POSTS_404_01 | Post not found. | 해당 `post_id` 게시글 없음 |
| 404 | COMMENTS_404_01 | Comment not found. | 해당 `comment_id` 댓글 없음 |
| 415 | COMMON_415_01 | Unsupported media type. | `Content-Type`이 `application/json` 아님 |
| 422 | COMMENTS_422_07 | Invalid content. | `content` 누락/빈값/정책 위반 |
| 422 | POSTS_422_07 | Invalid post_id. | `post_id` 형식 오류 |
| 422 | COMMENTS_422_08 | Invalid comment_id. | `comment_id` 형식 오류 |
