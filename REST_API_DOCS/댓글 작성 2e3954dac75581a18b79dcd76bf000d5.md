# 댓글 작성

리소스: Comments
method: POST
API path: comments, posts, {post_id}
Error: 400, 401, 404, 415, 422
비고: 작성된 게시글에 새로운 댓글을 답니다

### 댓글 작성

**POST** `/posts/{post_id}/comments`

작성된 게시글에 새로운 댓글을 답니다

**Request Headers**

| 헤더 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| Content-Type | application/json | ✅ | 텍스트를 전송하기 위한 포맷 |
| Authorization | string | ✅ | 댓글 작성을 하는 사람이 본인임을 확인하기 위한 토큰 (`Bearer <access_token>` 형식의 인증 토큰)(예: `Authorization: Bearer eyJ...`) |

**Path Parameters**

| 파라미터 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| `post_id` | string | ✅ | 댓글을 작성할 게시물의 고유 ID |

**Request Body**

| 필드 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| content | string | ✅ | 게시물의 본문 |

**Request Example** 

```json
{
  "content": "댓글 내용 1"
}
```

**Response (201 Created)**

```json
{
  "status": "success",
  "data": {
    "id": "comment_01",
    "post_id": "post_01",
    "content": "댓글 내용 1",
    "created_at": "2026-01-04T12:00:00Z",
    "author": {
      "id": "user_1234567abc",
      "nickname": "sihwa",
      "profile_image_url": "https://api.example.com/uploads/profiles/default.png"
    }
  }
}
```

**Error**

| HTTP | code | message | 발생 조건 |
| --- | --- | --- | --- |
| 400 | COMMON_400_01 | JSON parse failed. | `Authorization` 헤더 없음 |
| 401 | AUTH_401_01 | Authorization required. | 토큰 위조/형식 오류 |
| 401 | AUTH_401_02 | Invalid token. | 토큰 만료 |
| 401 | AUTH_401_03 | Token expired. | `Content-Type`이 `application/json` 아님 |
| 404 | POSTS_404_01 | Post not found. | 해당 `post_id` 게시글 없음 |
| 415 | COMMON_415_01 | Unsupported media type. | 서버 내부 오류 |
| 422 | POSTS_422_07 | Invalid post_id. | `post_id` 형식 오류 |
| 422 | COMMENTS_422_07 | Invalid content. | `content` 누락/빈값/정책 위반(길이 등) |
