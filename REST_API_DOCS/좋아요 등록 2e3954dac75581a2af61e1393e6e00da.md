# 좋아요 등록

리소스: Likes
method: POST
API path: likes, posts, {post_id}
Error: 401, 404, 409, 422
비고: 게시글에 좋아요를 누른다

### 좋아요 등록

**POST** `/posts/{post_id}/likes`

게시글에 좋아요를 누른다

**Request Headers**

| 헤더 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| Authorization | string | ✅ | 댓글 작성을 하는 사람이 본인임을 확인하기 위한 토큰 (`Bearer <access_token>` 형식의 인증 토큰)(예: `Authorization: Bearer eyJ...`) |

**Path Parameters**

| 파라미터 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| `post_id` | string | ✅ | 댓글을 작성할 게시물의 고유 ID |

**Response (201 Created)**

```json
{
  "status": "success",
  "data": {
    "post_id": "post_01",
    "liked": true,
    "like_count": 11
  }
}
```

**Error**

| HTTP | code | message | 발생 조건 |
| --- | --- | --- | --- |
| 401 | AUTH_401_01 | Authorization required. | `Authorization` 헤더 없음 |
| 401 | AUTH_401_02 | Invalid token. | 토큰 위조/형식 오류 |
| 401 | AUTH_401_03 | Token expired. | 토큰 만료 |
| 404 | POSTS_404_01 | Post not found. | 해당 `post_id` 게시글 없음 |
| 409 | LIKES_409_01 | Already liked. | 이미 좋아요가 등록된 상태 |
| 422 | POSTS_422_07 | Invalid post_id. | `post_id` 형식 오류 |
