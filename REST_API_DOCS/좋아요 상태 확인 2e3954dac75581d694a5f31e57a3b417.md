# 좋아요 상태 확인

리소스: Likes
method: GET
API path: likes, posts, {post_id}
Error: 401, 404, 422
비고: 현재 사용자가 보고 있는 게시물에 좋아요를  눌렀는지 확인합니다.
총 좋아요 수 또한 반환합니다.

### 좋아요 상태 확인

**GET** `/posts/{post_id}/likes`

현재 사용자가 보고 있는 게시물에 좋아요를  눌렀는지 확인합니다.

총 좋아요 수 또한 반환합니다.

**Request Headers**

| 헤더 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| Authorization | string | ✅ | 댓글 수정을 하는 사람이 본인임을 확인하기 위한 토큰 (`Bearer <access_token>` 형식의 인증 토큰)(예: `Authorization: Bearer eyJ...`) |

**Path Parameters**

| 파라미터 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| `post_id` | string | ✅ | 댓글이 속한 게시글의 고유 ID |

**Request Body**

**Request Example** 

```
GET /posts/post_01/likes
```

**Response (200 OK)**

```json
{
  "status": "success",
  "data": {
    "post_id": "post_01",
    "liked_by_me": true,
    "like_count": 10
  }
}
```

**Error**

| HTTP | code | message | 발생 조건 |
| --- | --- | --- | --- |
| 401 | AUTH_401_01 | Authorization required. | `Authorization` 헤더 없음 |
| 401 | AUTH_401_02 | Invalid token. | 토큰 위조/형식 오류 |
| 401 | AUTH_401_03 | Token expired. | 토큰 만료 |
| 404 | POSTS_404_01 | Post not found. | 해당 `post_id` 게시글 없음 |
| 422 | POSTS_422_07 | Invalid post_id. | `post_id` 형식 오류 |
