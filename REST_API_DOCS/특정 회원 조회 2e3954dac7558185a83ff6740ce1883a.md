# 특정 회원 조회

리소스: Users
method: GET
API path: users, {user_id}
Error: 404, 422
비고: user_id에 해당하는 사용자의 공개 프로필(닉네임, 프로필 이미지)을 반환한다

### 특정 회원 조회

**GET** `/users/{user_id}` 

user_id에 해당하는 사용자의 공개 프로필(닉네임, 프로필 이미지)을 반환한다

**Path Parameters**

| 파라미터 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| `user_id` | string | ✅ | 조회할 사용자의 고유 식별자 |

**Response (200 OK)**

```json
{
  "status": "success",
  "data": {
	  "id" : "user_1234567abc",
    "nickname": "sihwa2",
    "profile_image_url" : "https://api.example.com/uploads/profiles/default2.png"
  }
}
```

**Error**

| HTTP | code | 언제 나옴 |
| --- | --- | --- |
| 422 | USERS_422_05 | user_id 형식 오류(예: user_ prefix 규칙 등) |
| 404 | USERS_404_01 | 사용자 없음 |
