# 프로필 수정

리소스: Users
method: PATCH
API path: me, users
Error: 400, 401, 403, 409, 413, 415, 422
비고: 자신의 프로필을 수정합니다.

### 프로필 수정

**PATCH** `/users/me`

자신의 프로필을 수정합니다.

**Request Headers**

| 헤더 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| Content-Type | multipart/form-data | ✅ | 텍스트 필드와 파일(프로필 이미지)을 함께 전송하기 위한 포맷 |
| Authorization | string | ✅ | 수정을 하는 사람이 본인임을 확인하기 위한 토큰 (Bearer 스킴) |

**Request Body**

| 필드 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| current_password | string | ✅ | 본인 인증을 위한 패스워드 입력 |
| nickname | string | ❎ | users에 새로 등록할 닉네임 |
| profile_image | file | ❎ | users에 새로 등록할 프로필 사진 |
| new_password | string | ❎ | users에 새로 등록할 비밀번호 |

**Request Example  (multipart/form-data)**

| current_password | p@sswordex! |
| --- | --- |
| new_password | p@sswordnew! |
| nickname | My_name_change |
| profile_image | `profile.png`  / `profile.jpeg` / 최대 10mb |

**Response 200**

```json
{
  "status": "success",
  "data": {
	  "id" : "user_1234567abc",
    "nickname": "sihwa2",
    "profile_image_url" : "https://api.example.com/uploads/profiles/default2.png",
    "created_at": "2026-01-04T12:00:00Z"
  }
}
```

**Error**

| HTTP | code | 언제 나옴 |
| --- | --- | --- |
| 400 | COMMON_400_02 | multipart 파싱 실패 |
| 401 | AUTH_401_01~03 | 인증 관련(토큰 없음/무효/만료) |
| 403 | USERS_403_01 | current_password 불일치  |
| 409 | USERS_409_02 | nickname 중복 |
| 413 | UPLOAD_413_01 | profile_image 10MB 초과 |
| 415 | UPLOAD_415_01 | profile_image 타입 불일치(jpeg/png만) |
| 415 | COMMON_415_01 | Content-Type 불일치 |
| 422 | USERS_422_02 | new_password 정책 위반 |
