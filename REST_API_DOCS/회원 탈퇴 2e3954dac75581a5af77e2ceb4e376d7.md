# 회원 탈퇴

리소스: Users
method: DELETE
API path: me, users
Error: 400, 401, 403, 415, 422
비고: 로그인한 본인이 계정을 삭제합니다.

### 회원 탈퇴

**DELETE** `/users/me`

로그인한 본인이 계정을 삭제합니다.

**Request Headers**

| 헤더 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| Authorization | string | ✅ | 계정 삭제를 하는 사람이 본인임을 확인하기 위한 토큰 (`Bearer <access_token>` 형식의 인증 토큰)(예: `Authorization: Bearer eyJ...`) |

**Request Body**

| 필드 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| current_password | string | ✅ | 본인 인증을 위한 패스워드 입력 |
| confirm_text | string | ✅ | ‘DELETE’. 라는 입력값을 입력해야 탈퇴 진행 |

**Request Example** 

```json
{
  "current_password": "p@sswordex!",
  "confirm_text": "DELETE"
}
```

**Error**

| HTTP | code | 언제 나옴 |
| --- | --- | --- |
| 400 | COMMON_400_01 | JSON 파싱 실패 |
| 401 | AUTH_401_01~03 | 인증 관련 |
| 403 | USERS_403_01 | current_password 불일치 |
| 415 | COMMON_415_01 | Content-Type 불일치 |
| 422 | USERS_422_04 | confirm_text가 “DELETE”가 아님  |
