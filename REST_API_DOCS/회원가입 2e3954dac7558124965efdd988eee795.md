# 회원가입

리소스: Auth(인증)
method: POST
API path: auth, signup
Error: 400, 409, 413, 415, 422
비고: 새 사용자 계정을 생성한다(회원가입). 이메일, 비밀번호, 닉네임, 프로필 이미지(선택)를 입력해야 한다.

### 회원가입

**POST** `/auth/signup`

새 사용자 계정을 생성한다(회원가입). 이메일, 비밀번호, 닉네임, 프로필 이미지(선택)를 입력해야 한다.

**Request Headers**

| 헤더 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| Content-Type | multipart/form-data | ✅ | 텍스트 필드와 파일(프로필 이미지)을 함께 전송하기 위한 포맷 |

**Request Body**

| 필드 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| email | string | ✅ | 사용자가 사용할 @ 가 들어간 이메일 형식, 중복될 수 없다. |
| password | string | ✅ | 8이상 20자 이하의 문자열 조합.(공백 미 포함 영어만 사용 가능) 숫자와 영어소문자 특수문자는 각 1개씩 무조건 들어가야 한다. |
| nickname | string | ✅ | 사용자가 사용할 닉네임, 중복될 수 없다. 최대 20자 이하로만 생성 가능하다 |
| profile_image | file | ❎ | 바이너리 형식으로 받는 파일 10MB를 초과할 수 없다. jpeg/png파일을 받는다  |

**Request Example  (multipart/form-data)**

| email | 123psk@naver.com |
| --- | --- |
| password | p@sswordex! |
| nickname | My_name |
| profile_image | `profile.png`  / `profile.jpeg` / 최대 10mb |

**Response 201** 

```json
{
  "status": "success",
  "data": {
	  "id" : "user_1234567abc",
    "email": "sigwan116@gmail.com",
    "nickname": "sihwa",
    "profile_image_url" : "https://api.example.com/uploads/profiles/default.png",
    "created_at": "2026-01-04T12:00:00Z"
  }
}
```

**Note**

- `profile_image_url`: 프로필 이미지를 업로드하지 않은 경우 기본 이미지 URL(`.../default.png`)을 반환한다.

**Error**

| HTTP | code | 언제 나옴 |
| --- | --- | --- |
| 400 | COMMON_400_02 | multipart/form-data 파싱 실패 |
| 409 | USERS_409_01 | email 중복 |
| 409 | USERS_409_02 | nickname 중복 |
| 413 | UPLOAD_413_01 | profile_image 10MB 초과 |
| 415 | COMMON_415_01 | Content-Type이 multipart/form-data가 아님 |
| 415 | UPLOAD_415_01 | profile_image 확장자/타입 불일치(jpeg/png만 허용) |
| 422 | USERS_422_01 | email 형식 오류 |
| 422 | USERS_422_02 | password 정책 위반(길이/조합/공백 등) |
| 422 | USERS_422_03 | nickname 길이/형식 위반(최대 20자 등) |
